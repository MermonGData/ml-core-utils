A shared submodule containing standardized utility functions for machine learning workflows. Designed to keep notebook experimentation clean and production scripts consistent.

| Module | Purpose | Key Features |
| :--- | :--- | :--- |
| **`config.py`** | Environment Setup | Notebook initialization, plotting themes and global constants. |
| **`data_io.py`** | Data Handling | Standardized functions for loading/exporting raw and processed datasets. |
| **`evaluation.py`** | Model Assessment | Metric calculations, residual analysis, and performance visualization. |
| **`persistence.py`** | Artifact Management | Saving/loading models, metadata tracking, and bundling metrics for versioning. |
| **`preprocessing.py`** | Feature Engineering | Common cleaning routines and data transformation pipelines. |
